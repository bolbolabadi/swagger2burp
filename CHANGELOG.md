# Changelog

## v1.0.0 (2025-12-21)
- Initial public release of Swagger2Burp.
- Import Swagger 2.0 and OpenAPI 3 specs from URL or pasted JSON.
- Optional JWT/custom headers for spec fetch and generated requests.
- Base URL override; uses servers/basePath from spec when enabled.
- Auto-fill path/query parameters and generate example JSON bodies.
- Preview and send selected requests to Repeater.
